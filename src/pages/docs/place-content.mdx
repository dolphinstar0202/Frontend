## Basic usage

### Center

Use `place-content-center` to pack items in the center of the block axis:

<Example>
  <div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-sky rounded-lg">
    <div class="grid grid-cols-[repeat(2,56px)] place-content-center h-56 gap-4">
      <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-sky-500">01</div>
      <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-sky-500">02</div>
      <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-sky-500">03</div>
      <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-sky-500">04</div>
    </div>
  </div>
</Example>

```html
<div class="grid grid-cols-2 gap-4 **place-content-center** h-48 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
</div>
```

### Space between

Use `place-content-between` to distribute grid items along the block axis so that that there is an equal amount of space between each row on the block axis.

<Example>
  <div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-purple rounded-lg">
    <div class="grid grid-cols-[repeat(2,56px)] place-content-between h-56 gap-4">
      <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-purple-500">01</div>
      <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-purple-500">02</div>
      <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-purple-500">03</div>
      <div class="p-4 w-14 h-14 flex items-center justify-center shadow-lg rounded-lg bg-purple-500">04</div>
    </div>
  </div>
</Example>

```html
<div class="grid grid-cols-2 gap-4 **place-content-between** h-48 ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
</div>
```

### Stretch

Use `place-content-stretch` to stretch grid items along their grid areas on the block axis:

<Example>
  <div class="font-mono text-white text-sm font-bold leading-6 bg-stripes-pink rounded-lg">
    <div class="grid grid-cols-2 place-content-stretch h-56 gap-4">
      <div class="p-4 flex items-center justify-center shadow-lg rounded-lg bg-pink-500">01</div>
      <div class="p-4 flex items-center justify-center shadow-lg rounded-lg bg-pink-500">02</div>
      <div class="p-4 flex items-center justify-center shadow-lg rounded-lg bg-pink-500">03</div>
      <div class="p-4 flex items-center justify-center shadow-lg rounded-lg bg-pink-500">04</div>
