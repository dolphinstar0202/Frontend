## Basic usage

### Block & Inline

Use `inline`, `inline-block`, and `block` to control the flow of text and elements.

<Example p="none">
  <div class="mx-auto max-w-xs bg-white shadow-xl p-4 text-gray-500 text-sm leading-6 sm:text-base sm:leading-7">
    When controlling the flow of text, using the CSS property <span class="inline bg-sky-100 font-bold text-sm text-gray-900 font-mono rounded">display: inline</span>
    will cause the text inside the element to wrap normally.
    <br><br>
    While using the property
    <span class="inline-block bg-sky-100 font-bold text-sm text-gray-900 font-mono rounded">display: inline-block</span>
    will wrap the element to prevent the text inside from extending beyond its parent.
    <br><br>
    Lastly, using the property
    <span class="block bg-sky-100 font-bold text-sm text-gray-900 font-mono rounded">display: block</span>
    will put the element on it's own line and fill its parent.
  </div>
</Example>

```html
<div>
  When controlling the flow of text, using the CSS property
  <span class="**inline**">display: inline</span>
  will cause the text inside the element to wrap normally.

  While using the property <span class="**inline-block**">display: inline-block</span>
  will wrap the element to prevent the text inside from extending beyond its parent.

  Lastly, using the property <span class="**block**">display: block</span>
  will put the element on it's own line and fill its parent.
</div>
```

### Flow Root

Use `flow-root` to create a block-level element with its own [block formatting context](https://developer.mozilla.org/en-US/docs/Web/Guide/CSS/Block_formatting_context).

<Example p="none">
  <div class="mx-auto max-w-xs bg-white shadow-xl p-4 text-gray-500 text-sm leading-6 sm:text-base sm:leading-7">
    <div class="flow-root bg-stripes-purple mb-0.5">
      <div class="my-4 bg-white">Well, let me tell you something, funny boy. Y'know that little stamp, the one that says "New York Public Library"? Well that may not mean anything to you, but that means a lot to me. One whole hell of a lot.</div>
    </div>
    <div class="flow-root bg-stripes-purple mt-0.5">
      <div class="my-4 bg-white">Sure, go ahead, laugh if you want to. I've seen your type before: Flashy, making the scene, flaunting convention. Yeah, I know what you're thinking. What's this guy making such a big stink about old library books? Well, let me give you a hint, junior. </div>
    </div>
  </div>
</Example>

```html
<div class="p-4">
  <div class="**flow-root** ...">
    <div class="my-4 ...">Well, let me tell you something, ...</div>
  </div>
  <div class="**flow-root** ...">
    <div class="my-4 ...">Sure, go ahead, laugh if you want...</div>
  </div>
</div>
```

### Flex

Use `flex` to create a block-level flex container.

<Example p="none">
  <div class="mx-auto max-w-xs bg-white shadow-xl p-4 text-gray-500 leading-6 flex justify-center">
    <div class="flex items-center gap-4 p-4">
      <img class="w-12 h-12 rounded-full" src="https://images.unsplash.com/photo-1501196354995-cbb51c65aaea?ixlib=rb-1.2.1&amp;ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&amp;auto=format&amp;fit=facearea&amp;facepad=4&amp;w=256&amp;h=256&amp;q=80">
      <div class="flex flex-col">
        <strong class="text-gray-900 font-medium">Andrew Alfred</strong>
        <span class="text-gray-500 font-medium">Technical advisor</span>
      </div>
    </div>
  </div>
</Example>

```html
<div class="**flex** items-center">
  <img src="path/to/image.jpg">
  <div>
    <strong>Andrew Alfred</strong>
    <span>Technical advisor</span>
  </div>
</div>
```

### Inline Flex

Use `inline-flex` to create an inline flex container that flows with text.

<Example p="none">
  <div class="mx-auto max-w-lg bg-white shadow-xl p-4 text-gray-500 text-sm leading-6 sm:text-base sm:leading-7">
    No. It isn't and the most distressing part of it is, not that she went out with him but that HE stopped seeing her. Do you understand? He,
    <div class="inline-flex items-center align-middle mt-[-3px]">
      <img src="https://images.unsplash.com/photo-1501196354995-cbb51c65aaea?ixlib=rb-1.2.1&amp;ixid=MnwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8&amp;auto=format&amp;fit=facearea&amp;facepad=4&amp;w=256&amp;h=256&amp;q=80" alt="" class="w-5 h-5 rounded-full mx-1" />
      <span class="font-medium text-gray-900">Newman</span>
    </div>,
    stopped seeing her. Newman never stopped seeing anybody. Newman will see whoever is willing to see him. Not so much why she did see him as disturbing as that is. But why, did HE, Newman, stop seeing her?
  </div>
</Example>

```html
<div>
  No. It isn't and the most distressing part of it is, ...
  <div class="**inline-flex** items-center align-middle">
    <img src="path/to/image.jpg" alt="" class="w-5 h-5 rounded-full mx-1" />
    <span>Newman</span>
  </div>
  stopped seeing her. Newman never stopped seeing anybody. ...
</div>
```

### Contents

Use `contents` to create a "phantom" container whose children act like direct children of the parent.

<Example>
  <div class="flex gap-4 font-mono text-white text-sm font-bold leading-6 bg-stripes-purple rounded-lg text-center">
    <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">01</div>
    <div class="contents">
      <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">02</div>
      <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">03</div>
    </div>
    <div class="flex-1 p-4 bg-purple-500 shadow-lg rounded-lg">04</div>
  </div>
</Example>

```html
<div class="flex ...">
  <div class="flex-1 ...">01</div>
  <div class="contents">
    <div class="flex-1 ...">02</div>
    <div class="flex-1 ...">03</div>
  </div>
  <div class="flex-1 ...">04</div>
</div>
```

### Hidden

Use `hidden` to set an element to `display: none` and remove it from the page layout (compare with `.invisible` from the [visibility](/docs/visibility#invisible) documentation).

<Example>
  <div class="flex gap-4 font-mono text-white text-sm font-bold leading-6 bg-stripes-purple rounded-lg text-center">
    <div class="hidden p-4 w-14 h-14 bg-purple-500 shadow-lg rounded-lg">01</div>
    <div class="p-4 w-14 h-14 bg-purple-500 shadow-lg rounded-lg">02</div>
    <div class="p-4 w-14 h-14 bg-purple-500 shadow-lg rounded-lg">03</div>
  </div>
</Example>

```html
<div class="flex ...">
  <div class="**hidden** ...">01</div>
  <div>02</div>
  <div>03</div>
</div>
```
