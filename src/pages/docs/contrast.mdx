## Using custom values

### Customizing your theme

By default, Tailwind includes a handful of general purpose `contrast` utilities. You can customize these values by editing `theme.contrast` or `theme.extend.contrast` in your `tailwind.config.js` file.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       contrast: {
+         25: '.25',
+       }
      }
    }
  }
```

Learn more about customizing the default theme in the [theme customization](/docs/theme#customizing-the-default-theme) documentation.
