## Basic usage

### Sizing implicitly-created grid rows

Use the `auto-rows-{size}` utilities to control the size implicitly-created grid rows.

```html
<div class="grid grid-flow-row **auto-rows-max**">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

## Using custom values

### Customizing your theme

By default, Tailwind includes four general purpose `grid-auto-rows` utilities. You can customize these values by editing `theme.gridAutoRows` or `theme.extend.gridAutoRows` in your `tailwind.config.js` file.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       gridAutoRows: {
+         '2fr': 'minmax(0, 2fr)',
+       }
      }
    }
  }
```

Learn more about customizing the default theme in the [theme customization](/docs/theme#customizing-the-default-theme) documentation.
