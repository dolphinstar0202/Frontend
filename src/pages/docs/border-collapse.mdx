## Basic usage

### Collapse

Use `border-collapse` to combine adjacent cell borders into a single border when possible. Note that this includes collapsing borders on the top-level `<table>` tag.

<Example p="none">
  <div class="px-4 py-8 sm:px-8">
    <table class="border-collapse w-full border border-gray-400 bg-white text-sm shadow-sm">
      <thead class="bg-gray-50">
        <tr>
          <th class="w-1/2 border border-gray-300 font-semibold p-4 text-gray-900 text-left">State</th>
          <th class="w-1/2 border border-gray-300 font-semibold p-4 text-gray-900 text-left">City</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td class="border border-gray-300 p-4 text-gray-500">Indiana</td>
          <td class="border border-gray-300 p-4 text-gray-500">Indianapolis</td>
        </tr>
        <tr>
          <td class="border border-gray-300 p-4 text-gray-500">Ohio</td>
          <td class="border border-gray-300 p-4 text-gray-500">Columbus</td>
        </tr>
        <tr>
          <td class="border border-gray-300 p-4 text-gray-500">Michigan</td>
          <td class="border border-gray-300 p-4 text-gray-500">Detroit</td>
        </tr>
      </tbody>
    </table>
  </div>
</Example>

```html
<table class="**border-collapse** border border-gray-400 ...">
  <thead>
    <tr>
      <th class="border border-gray-300 ...">State</th>
      <th class="border border-gray-300 ...">City</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td class="border border-gray-300 ...">Indiana</td>
      <td class="border border-gray-300 ...">Indianapolis</td>
    </tr>
    <tr>
      <td class="border border-gray-300 ...">Ohio</td>
      <td class="border border-gray-300 ...">Columbus</td>
    </tr>
    <tr>
      <td class="border border-gray-300 ...">Michigan</td>
      <td class="border border-gray-300 ...">Detroit</td>
    </tr>
  </tbody>
</table>
```

### Separate

Use `border-separate` to force each cell to display its own separate borders.

<Example p="none">
  <div class="px-4 py-8 sm:px-8">
    <table class="border-separate w-full border border-gray-400 bg-white text-sm shadow-sm">
      <thead class="bg-gray-50">
        <tr>
          <th class="w-1/2 border border-gray-300 font-semibold p-4 text-gray-900 text-left">State</th>
          <th class="w-1/2 border border-gray-300 font-semibold p-4 text-gray-900 text-left">City</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <td class="border border-gray-300 p-4 text-gray-500">Indiana</td>
          <td class="border border-gray-300 p-4 text-gray-500">Indianapolis</td>
        </tr>
        <tr>
          <td class="border border-gray-300 p-4 text-gray-500">Ohio</td>
          <td class="border border-gray-300 p-4 text-gray-500">Columbus</td>
        </tr>
        <tr>
          <td class="border border-gray-300 p-4 text-gray-500">Michigan</td>
          <td class="border border-gray-300 p-4 text-gray-500">Detroit</td>
        </tr>
      </tbody>
    </table>
  </div>
</Example>
