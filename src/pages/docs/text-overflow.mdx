## Truncate

Use `truncate` to truncate overflowing text with an ellipsis (<code>…</code>) if needed.

```html
<template preview>
  <p class="max-w-xs p-2 mx-auto truncate bg-gray-200">
  Lorem ipsum dolor sit amet, consectetur adipisicing elit. Blanditiisitaquequodpraesentiumexplicaboincidunt? Dolores beatae nam at sed dolorum ratione dolorem nisi velit cum.
  </p>
</template>

<p class="truncate ...">...</p>
```

## Overflow Clip

Use `overflow-clip` to truncate the text at the limit of the content area.

```html
<template preview>
  <p class="max-w-xs p-2 mx-auto bg-gray-200 overflow-clip">
  Lorem ipsum dolor sit amet, consectetur adipisicing elit. Blanditiisitaquequodpraesentiumexplicaboincidunt? Dolores beatae nam at sed dolorum ratione dolorem nisi velit cum.
  </p>
</template>

<p class="overflow-clip ...">...</p>
```

## Responsive

To control the text overflow in an element only at a specific breakpoint, add a `{screen}:` prefix to any existing text overflow utility. For example, adding the class `md:truncate` to an element would apply the `truncate` utility at medium screen sizes and above.

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.

<ResponsiveCodeSample
  classNames={['truncate', 'overflow-ellipsis', 'overflow-clip']}
  snippet={(classNames) => `
    <p class="${classNames} ...">
      ...
    </p>
  `}
  preview={(className) => (
    <p className={`max-w-xs p-2 bg-gray-200 mx-auto ${className}`}>
      Lorem ipsum dolor sit amet, consectetur adipisicing elit. Blanditiisitaquequodpraesentiumexplicaboincidunt? Dolores beatae nam at sed dolorum ratione dolorem nisi velit cum.
    </p>
  )}
/>
