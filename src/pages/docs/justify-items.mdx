## Basic usage

### Start

Use `justify-items-start` to justify grid items against the start of their inline axis:

<Example>
  <div class="grid grid-cols-3 gap-4 font-mono text-white text-sm font-bold leading-6">
    <div class="bg-stripes-sky grid rounded-lg justify-items-start">
      <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">01</div>
    </div>
    <div class="bg-stripes-sky grid rounded-lg justify-items-start">
      <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">02</div>
    </div>
    <div class="bg-stripes-sky grid rounded-lg justify-items-start">
      <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">03</div>
    </div>
    <div class="bg-stripes-sky grid rounded-lg justify-items-start">
      <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">04</div>
    </div>
    <div class="bg-stripes-sky grid rounded-lg justify-items-start">
      <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">05</div>
    </div>
    <div class="bg-stripes-sky grid rounded-lg justify-items-start">
      <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-sky-500 shadow-lg">06</div>
    </div>
  </div>
</Example>

```html
<div class="grid **justify-items-start** ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
  <div>05</div>
  <div>06</div>
</div>
```

      <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-cyan-500 shadow-lg">02</div>

    <div class="bg-stripes-cyan grid rounded-lg justify-items-end">
      <div class="w-14 h-14 rounded-lg flex items-center justify-center bg-cyan-500 shadow-lg">03</div>

Use `justify-items-center` to justify grid items along their inline axis:

### Stretch

Use `justify-items-stretch` to stretch items along their inline axis:

<Example>
  <div class="grid grid-cols-3 gap-4 justify-items-stretch font-mono text-white text-sm font-bold leading-6">
    <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">01</div>
    <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">02</div>
    <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">03</div>
    <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">04</div>
    <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">05</div>
    <div class="h-14 rounded-lg flex items-center justify-center bg-blue-500 shadow-lg">06</div>
  </div>
</Example>

```html
<div class="grid **justify-items-stretch** ...">
  <div>01</div>
  <div>02</div>
  <div>03</div>
  <div>04</div>
  <div>05</div>
  <div>06</div>
</div>
```
