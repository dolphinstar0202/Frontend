## Using custom values

### Customizing your theme

By default, Tailwind includes a handful of general purpose `blur` utilities. You can customize these values by editing `theme.blur` or `theme.extend.blur` in your `tailwind.config.js` file.

```diff-js tailwind.config.js
  module.exports = {
    theme: {
      extend: {
+       blur: {
+         xs: '2px',
+       }
      }
    }
  }
```

Learn more about customizing the default theme in the [theme customization](/docs/theme#customizing-the-default-theme) documentation.
