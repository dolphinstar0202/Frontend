## Basic usage

### Auto

Use `table-auto` to allow the table to automatically size columns to fit the contents of the cell.

<Example p="none">
  <div class="shadow-sm overflow-hidden my-8">
    <table class="border-collapse table-auto w-full text-sm">
      <thead>
        <tr>
          <th class="border-b font-medium p-4 pl-8 pt-0 pb-3 text-gray-400 text-left">Song</th>
          <th class="border-b font-medium p-4 pt-0 pb-3 text-gray-400 text-left">Artist</th>
          <th class="border-b font-medium p-4 pr-8 pt-0 pb-3 text-gray-400 text-left">Year</th>
        </tr>
      </thead>
      <tbody class="bg-white">
        <tr>
          <td class="border-b border-gray-100 p-4 pl-8 text-gray-500">The Sliding Mr. Bones (Next Stop, Pottersville)</td>
          <td class="border-b border-gray-100 p-4 text-gray-500">Malcolm Lockyer</td>
          <td class="border-b border-gray-100 p-4 pr-8 text-gray-500">1961</td>
        </tr>
        <tr>
          <td class="border-b border-gray-100 p-4 pl-8 text-gray-500">Witchy Woman</td>
          <td class="border-b border-gray-100 p-4 text-gray-500">The Eagles</td>
          <td class="border-b border-gray-100 p-4 pr-8 text-gray-500">1972</td>
        </tr>
        <tr>
          <td class="border-b border-gray-200 p-4 pl-8 text-gray-500">Shining Star</td>
          <td class="border-b border-gray-200 p-4 text-gray-500">Earth, Wind, and Fire</td>
          <td class="border-b border-gray-200 p-4 pr-8 text-gray-500">1975</td>
        </tr>
      </tbody>
    </table>
  </div>
</Example>

```html

      <th>Artist</th>
      <th>Year</th>
```

### Fixed

Use `table-fixed` to allow the table to ignore the content and use fixed widths for columns. The width of the first row will set the column widths for the whole table.

You can manually set the widths for some columns and the rest of the available width will be divided evenly amongst the columns without explicit width.

<Example p="none">
  <div class="shadow-sm overflow-hidden my-8">
    <table class="border-collapse table-fixed w-full text-sm">
      <thead>

          <th class="border-b font-medium p-4 pl-8 pt-0 pb-3 text-gray-400 text-left">Song</th>
          <th class="border-b font-medium p-4 pt-0 pb-3 text-gray-400 text-left">Artist</th>
          <th class="border-b font-medium p-4 pr-8 pt-0 pb-3 text-gray-400 text-left">Year</th>
        </tr>
      </thead>
      <tbody class="bg-white">

          <td class="border-b border-gray-100 p-4 pl-8 text-gray-500">The Sliding Mr. Bones (Next Stop, Pottersville)</td>
          <td class="border-b border-gray-100 p-4 text-gray-500">Malcolm Lockyer</td>
          <td class="border-b border-gray-100 p-4 pr-8 text-gray-500">1961</td>
        </tr>
        <tr>
          <td class="border-b border-gray-100 p-4 pl-8 text-gray-500">Witchy Woman</td>

          <td class="border-b border-gray-100 p-4 pr-8 text-gray-500">1972</td>
