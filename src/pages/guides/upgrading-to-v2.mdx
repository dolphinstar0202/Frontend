## Upgrading

- Update your build environment to a modern version of Node
- Tell any customers who use IE11 to get with the times
- Don't depend on default colors
- Override the `fontSize` config or add line-height utilities to your templates to override the new default line-height values per font-size
- Add any extra variants you need for `truncate` to the `textOverflow` plugin
- Remove `scrolling-touch` and `scrolling-auto` from your HTML, they don't do anything
- Rename `whitespace-no-wrap`, `flex-no-wrap`, and `col/row-gap` utilities
- Update use of `theme` to be less data structure aware
- Add `hidden` to any `template` elements that are within `space` or `divide` blocks
- Disable `preserveHtmlElements` if you're using a custom extractor and don't want it respected
- Keyframes are prefixed automatically if you have a prefix in your config, make sure any custom CSS referencing those keyframe names accounts for this
- If you were relying on `extends` only merging shallowly, you will have to move those customizations out of `extends`
- Talk about upgrading custom forms and typography plugins?
